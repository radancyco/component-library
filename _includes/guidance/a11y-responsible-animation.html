Responsible Animation

Be mindful of the impact that animation can have on people with disabilities. Animation can be a distraction for some, make others ill, and dangerous for those subject to seizures. By sticking to the following guidelines, we can make our web experiences pleasant and safe for everybody!

  1. Any web animation that loops must include a pause button. There are no exceptions here.
  2. Animations that occur once when interacting with a web page are exempt from having a pause button if they do not last for more than 5 seconds and are not excessive in their motion. However, developers should always exercise caution and honor user settings by including prefers-reduced-motion to help reduce or stop animation where needed. Please visit the Mozilla Developer Network for more details about this useful media query.
  3. Animations that flash more than 3 times per second are not allowed on our websites, as they may induce seizures. A pause button will be ineffective here. If you suspect that an animation may be dangerous, please bring this to the attention of the Accessibility Team.
